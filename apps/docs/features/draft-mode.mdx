description: "Preview unpublished product pages with Draft Mode"

---

Draft Mode allows you to preview unpublished Shopify product data on your live website, rendered with all the usual styling and layout. This enables safe and efficient testing and collaboration before making changes public.

<Frame>
  <img className="block" src="/images/draft-toolbar.png" alt="Draft toolbar" />
</Frame>

**Enabling Draft Mode:**

1. Append `/draft` to the end of the product page URL.
2. Example: `/products/adidas-classic-backpack-legend-ink-multicolour/draft`

**Technical Details:**

Draft Mode leverages the Vercel Toolbar for workflow collaboration: https://vercel.com/docs/workflow-collaboration/vercel-toolbar
